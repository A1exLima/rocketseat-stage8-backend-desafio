## 👨‍💻 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Java Script
- SQL
- NodeJS
- SQLite
- Knex
- Express
- Express async errors
- Date fns
- Cors
- JWT (jsonwebtoken)
- Multer
- Bcrypt
- Git e Github
- Insomnia (testes de rotas)
- Beekeeper (Visualização Banco de Dados)
